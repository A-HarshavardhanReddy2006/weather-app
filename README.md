# Weather App

A modern weather web application built using FastAPI, HTML, CSS, and OpenWeather API.

## Features
- Real-time weather updates
- Beautiful UI animations
- Recent search history
- Responsive design
- Error handling for invalid city names

## Technologies Used
- Python
- FastAPI
- HTML
- CSS
- JavaScript
- OpenWeather API

## Project Structure

```bash
weather_project/
│
├── static/
│   └── style.css
│
├── templates/
│   └── index.html
│
├── main.py
└── .gitignore
```

## Run Locally

```bash
pip install fastapi uvicorn requests jinja2
uvicorn main:app --reload
```
